# 🎯 Sniper Desk

ICT/SMC trading assistant for GBP/USD, XAU/USD, and AUD/USD.
Built for Philippine timezone traders.

**🔗 Live app:** https://Kaikeeshi090825.github.io/sniper-desk/

## What it does

- Multi-timeframe bias board (Daily / 4H / 1H)
- Killzone clock in Manila time
- ICT Fibonacci OTE reference
- Fundamental analysis with red/orange/yellow folder weighting
- Tech vs Fund composite verdict — tells you whether to take the trade
- Risk and R:R calculator
- Trade journal with persistent storage
- Performance stats
- Printable daily cheat sheet

## How to use

Open the link, mark your bias, log fundamental events, 
get a verdict telling you whether to go long, short, or skip.

Your data saves in your browser locally — totally private.
Different browser or device = different data, so use the same one.

## Built for

Scalping and intraday trading in PHT during London KZ (3-6 PM) 
and NY KZ (8:30-11 PM).
